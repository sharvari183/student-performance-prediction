# 🎓 Student Performance Prediction System

## 📌 Overview
This project is a Machine Learning-based system that predicts a student's final academic performance using key behavioral and academic features such as study hours, attendance, quiz scores, assignment scores, and midterm marks.

The goal is to demonstrate how data-driven models can be used in education to identify student performance trends and support early intervention strategies.

---

## 🎯 Objective
- Predict student final score using ML
- Analyze relationship between study habits and performance
- Build an end-to-end machine learning workflow
- Demonstrate real-world data science project structure

---

## 📊 Dataset Features
- Study Hours
- Attendance Percentage
- Quiz Average
- Assignment Average
- Midterm Marks
- Final Score (Target)

---

## 🤖 Machine Learning Model
- Algorithm: Linear Regression
- Library: Scikit-learn
- Approach: Supervised Learning (Regression)

---

## 🧠 Workflow
Data Collection → Data Preprocessing → Model Training → Prediction → Evaluation

---

## 📈 Model Output
The model predicts the **final score of a student** based on input academic features.
